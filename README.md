# 🤖 AI System Comparator Dashboard

## 📌 Overview

An interactive dashboard built using Streamlit to compare and evaluate multiple AI systems (ChatGPT, Watsonx, Perplexity, DeepSeek) based on user queries and performance metrics.

---

## 🎯 Objective

To simulate how organizations evaluate different AI models by comparing their responses, tracking user feedback, and analyzing performance across multiple dimensions.

---

## ⚙️ Features

* 🔍 Multi-system AI response comparison
* 🧠 User query input & prebuilt prompts
* 👍👎 Feedback system for evaluation
* 📊 Performance visualization (Bar charts)
* 📈 Radar chart for multi-dimensional analysis
* 🏆 Recommendation system for best-performing model

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Plotly
* Pandas

---

## 📊 Key Insights

* Enables side-by-side comparison of LLM outputs
* Demonstrates AI evaluation workflow used in real-world systems
* Helps identify best-performing models for different use cases

---

## 🚀 How to Run Locally

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
